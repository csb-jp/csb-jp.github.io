静的ファイルのアップロード

画像や音声などのテキスト以外のファイルをアップロードする方法を説明します。左側のファイル一覧のエリアにある[Upload Files]ボタンを選択し、アップロードするファイルを選択します。

制約事項#

アップロードできるファイルのサイズは通常 20MB までです。
